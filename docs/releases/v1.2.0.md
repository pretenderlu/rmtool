# rmtool v1.2.0

## English

This release adds a full user-font manager and safer tap-to-turn deployment for
devices that already use Vellum/AppLoader.

### Font management

- Browse top-level TTF/OTF files in the device user-font directory and see the
  internal family and active status.
- Upload multiple fonts while keeping the original filename by default; the
  optional `zwzt.ttf` rename is still available.
- Set an existing font as the system UI font, delete inactive fonts, refresh
  the inventory, and reboot through separate explicit actions.
- Bind Fontconfig to the exact selected file, fixing cases where differently
  named files share the same internal family and the old font was selected.
- Protect active, system, and localization-managed fonts. Upload, activation,
  and deletion use validation and transactional rollback.

### Tap to turn pages

- Install tap-to-turn as a local Vellum APK when Vellum owns the standard
  AppLoader/Xovi runtime; clean devices continue to use rmtool's standalone
  deployment.
- Verify package ownership, runtime hashes, firmware, hardware, xochitl, QMD,
  and APK contents before mutation.
- Refuse conflicting or modified Xovi installations instead of merging unknown
  persistence settings.
- The Vellum package has no AppLoader icon or on-device toggle. While installed,
  its QMD loads whenever Xovi and qt-resource-rebuilder are active.
- Enable and disable never restart xochitl or reboot the device automatically.

### Notes

- Restart the device with the dedicated font-manager action after selecting a
  new system font.
- Some AppLoader installations require Xovi to be activated manually after a
  reboot before tap-to-turn becomes active.
- Tap-to-turn remains restricted to exact supported device and firmware
  combinations.

## 中文

本版本增加完整的用户字体管理器，并为已经使用 Vellum/AppLoader 的设备提供更安全的点击翻页部署方式。

### 字体管理

- 浏览设备用户字体目录顶层的 TTF/OTF，显示内部字体族和当前使用状态。
- 支持保存多个字体；上传默认保留原文件名，仍可主动选择重命名为 `zwzt.ttf`。
- 上传、设置系统字体、删除未使用字体、刷新列表和重启生效均为独立操作。
- Fontconfig 精确绑定所选文件路径，修复多个不同文件名共享同一内部字体族时仍选中旧字体的问题。
- 保护当前字体、系统字体和汉化内部字体；上传、切换和删除均带校验及事务回滚。

### 点击翻页

- 标准 AppLoader/Xovi 由 Vellum 管理时，以本地生成的 Vellum APK 安装点击翻页；干净设备继续使用 rmtool 独立部署。
- 修改前精确校验包所有权、运行库哈希、固件、硬件、xochitl、QMD 和 APK 内容。
- 遇到冲突包、自定义或被修改的 Xovi 时拒绝部署，不自动合并未知持久化配置。
- Vellum 包不增加 AppLoader 图标或设备端开关；安装期间 QMD 会在 Xovi 与 qt-resource-rebuilder 激活时始终加载。
- 启用和停用不会自动重启 xochitl 或设备。

### 使用提示

- 设置新的系统字体后，请使用字体管理器中的独立“重启生效”操作。
- 部分 AppLoader 安装在设备重启后需要手动激活 Xovi，点击翻页才会生效。
- 点击翻页仍只允许设备型号和固件版本精确匹配的受支持组合。
