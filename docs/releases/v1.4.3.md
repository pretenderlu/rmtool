# rmtool v1.4.3

## English

> Firmware 3.28.0.163 beta support for localization and tap-to-turn, plus
> reliability fixes for sleep-screen carousel blanking and the localization
> backup flow.

### Firmware 3.28.0.163 beta

- Chinese UI localization now supports beta `3.28.0.163` (internal version
  `20260702125656`) on Paper Pro (Ferrari) and Paper Pro Move (Chiappa).
  The stock translation catalogs are byte-identical to `3.28.0.162`, so the
  existing shared catalog is reused. Device-verified on Paper Pro.
- Tap-to-turn packages were rebuilt against the new `xochitl` binaries for
  both platforms; QML recovery, hashtable generation, and compatibility
  checks all pass. Device-verified on Paper Pro.

### Wallpaper

- Fixed the sleep-screen carousel alternating between original illustrations
  and transparent frames on newer firmware: carousel backups were stored
  next to the originals, but the firmware rotates every file in the carousel
  directory regardless of extension. Backups now live in the
  `carousel/.backup/` subdirectory, which the firmware ignores, and legacy
  side-by-side backups are migrated automatically.
- The wallpaper orientation row is hidden and fixed to portrait; landscape
  logic remains in place for later re-enabling.

### Localization reliability

- The enable/restore/backup flows now sync device writes before reporting
  success. A hard power-off within the ext4 writeback window previously left
  zero-byte backup files, which then blocked localization with hash-mismatch
  errors.
- When the on-device carrier is verified stock, an unusable backup (zeroed
  files, or a backup from an older firmware) is discarded instead of
  blocking every operation; the next enable rebuilds a fresh backup.
  Validation remains strict whenever the carrier is localized.

## 中文

> 支持 3.28.0.163 测试版固件的汉化与点击翻页，并修复休眠轮播空白化和
> 汉化备份流程的可靠性问题。

### 3.28.0.163 测试版固件

- 中文界面汉化支持 `3.28.0.163`（内部版本 `20260702125656`),覆盖
  Paper Pro(Ferrari）与 Paper Pro Move(Chiappa)。官方翻译目录与
  `3.28.0.162` 逐字节相同，复用现有共享中文目录。已在 Paper Pro 真机
  验证。
- 点击翻页包针对新版 `xochitl` 二进制重新构建,QML 资源恢复、哈希表
  生成与兼容性检查全部通过。已在 Paper Pro 真机验证。

### 壁纸

- 修复新固件休眠轮播"原图/透明"交替显示的问题：轮播备份原先与原图
  放在同一目录，而固件会遍历目录内所有文件（不看扩展名）。备份改存
  固件会忽略的 `carousel/.backup/` 子目录，旧版并排备份自动迁移。
- 壁纸方向设置行已隐藏并固定为竖屏，横屏逻辑保留以便日后恢复。

### 汉化可靠性

- 启用/恢复/备份流程在报告成功前强制落盘（sync)。此前在 ext4 回写
  窗口内强制断电会产生 0 字节备份文件，导致汉化被哈希不匹配错误卡死。
- 当设备上的载体已确认为官方原版时，无法使用的备份（0 字节文件或
  旧固件遗留）会被自动丢弃并按未安装处理，重新启用即可重建备份；
  载体为中文版时校验保持严格。
